# Changelog

## 0.0.6

- Start with a bundled demo workspace so the extension is useful immediately after install.
- Keep real Weaviate connections optional through the connection panel.

## 0.0.5

- Add a Weaviate DB Activity Bar view for VS Code-compatible editors such as Windsurf.
- Keep the existing command palette action for opening the workbench in a full editor tab.

## 0.0.4

- Improve Marketplace metadata and user-facing README.
- Add a Marketplace icon.

## 0.0.3

- Bundle the Weaviate workbench as a VS Code-compatible webview extension.
- Support installing the same VSIX in VS Code-compatible editors, including Windsurf.
