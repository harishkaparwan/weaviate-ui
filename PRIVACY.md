# Privacy Policy

Weaviate DB does not collect, store, sell, or share user data.

The extension runs locally in the browser and is used to connect to Weaviate endpoints that the user provides. Requests are sent directly from the user's browser to the configured Weaviate endpoint.

The extension does not transmit data to the developer or to any third-party analytics, advertising, or tracking service.

Data entered into the extension, including endpoint URLs, JSON, CSV content, and GraphQL queries, remains in the user's browser session unless sent by the user to their configured Weaviate endpoint.

Host permissions are required so users can connect to their own local or remote Weaviate HTTP endpoints. The extension only sends requests to endpoints configured or used by the user.

No remote code is used. The extension bundles its UI assets and runs them locally.
